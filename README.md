# PureBloom GitHub Pages Website

## Upload to GitHub

1. Open your GitHub repository.
2. Click **Add file → Upload files**.
3. Upload:
   - `index.html`
   - `style.css`
   - `script.js`
   - the complete `images` folder
4. Click **Commit changes**.
5. Go to **Settings → Pages**.
6. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
7. Click **Save**.

## Replace the placeholders

### Calendly
Open `index.html` and replace every occurrence of:

`https://calendly.com/YOUR-LINK`

with your actual Calendly URL.

### YouTube
Replace:

`https://www.youtube.com/embed/VIDEO_ID`

with the embed URL of your uploaded YouTube video.

Example:

`https://www.youtube.com/embed/abc123xyz`
